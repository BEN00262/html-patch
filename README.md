### html-patche v1.0.9

#### installation
```bash
$ npm i -g html-patche
```

#### initializing a project
```bash
$ html-patche -i sample-project
$ cd sample-project
```

#### running a dev server
```bash
$ html-patche -d
```

#### building the project
```bash
$ html-patche -b # the website is saved ina build folder
```
